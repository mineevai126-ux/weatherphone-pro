# Serz v3 build notes

This build connects the new Serz dashboard as the launcher activity.

Included:

- exact-location forecast entry point;
- favorite cities storage;
- forecast accuracy memory;
- provider comparison card;
- scenario advice cards;
- tabs: Now, Accuracy, Scenarios, 7 days;
- Android launcher points to SerzActivity.

Expected APK artifact name: app-debug.apk.
