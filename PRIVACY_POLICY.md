# Политика конфиденциальности — Суперпрогноз от Serz

Дата: 28 апреля 2026

Приложение «Суперпрогноз от Serz» показывает прогноз погоды, анализирует погодные риски и помогает пользователю принимать бытовые решения: брать ли зонт, как одеться, можно ли идти на прогулку, насколько надёжен прогноз.

## Какие данные использует приложение

Приложение может использовать:

- название города, введённое пользователем;
- приблизительную или точную геолокацию, если пользователь сам дал разрешение;
- погодные данные из внешних погодных API;
- локальные настройки приложения;
- локальную историю прогнозов для оценки точности.

## Геолокация

Геолокация используется только для построения прогноза по точной точке пользователя. Приложение не требует геолокацию постоянно и не передаёт её владельцу приложения.

## Локальное хранение

Приложение может сохранять на устройстве:

- избранные города;
- город по умолчанию;
- локальную память точности прогноза;
- данные для виджета.

Эти данные хранятся локально на устройстве пользователя.

## Передача данных третьим лицам

Для получения прогноза приложение обращается к погодным сервисам, например Open-Meteo. В такие сервисы может передаваться город или координаты точки прогноза.

## Уведомления

Приложение может отправлять уведомления о погодных рисках, например о возможном дожде. Уведомления работают только после разрешения пользователя.

## Реклама и аналитика

В текущей версии приложение не содержит рекламных SDK и не подключает стороннюю аналитику.
